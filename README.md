# 🔒 LinkTrust - Advanced Link Safety Analyzer

**LinkTrust** is a powerful real-time link analyzer that helps users detect phishing attempts, hidden redirects, and potentially malicious URLs **before** they click them.

### 🌟 Features

- **Real Redirect Tracking** — Follows and displays the full redirect chain
- **Risk Assessment** — Gives a clear safety score (Low / Medium / High Risk)
- **Domain Analysis** — Checks for suspicious shorteners and risky domains
- **Clean & Modern UI** — Beautiful, responsive design that works on mobile & desktop
- **Fast & Lightweight** — Built with pure HTML, CSS & JavaScript (no backend required for demo)

### 🛠️ How It Works

1. Paste any link (bit.ly, t.co, normal URLs, etc.)
2. Click **Analyze**
3. Get instant report showing:
   - Redirect chain
   - Final destination
   - Risk level
